# Rust-like Language Compiler Web

A lightweight web project for compiling a Rust-like language, built with **Flask**.

## Highlights

- **Rust-like syntax support**: experiment with language design inspired by Rust.
- **Compiler workflow visualization**: clearly show core compilation stages.
- **Web-based experience**: write and compile in browser without local setup.
- **Flask backend + classic frontend**: simple architecture, easy to understand and extend.
- **Great for learning**: suitable for compiler courses, demos, and personal language projects.

## Project Structure

- `app.py` / Flask app entry
- `templates/` for HTML pages
- `static/` for frontend assets
- compiler-related modules(.py files) for lexical/syntax/semantic or intermediate processing

## Quick Start

1. Clone the repository:

2. Install dependencies and run the server:
   ```bash
   pip install -r requirements.txt
   python app.py
   ```

3. Open in browser:
   - `http://127.0.0.1:5000`
